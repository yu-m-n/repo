# 🔗 통합 프로젝트

이 디렉토리는 팀원들의 개인 작업 결과를 **통합**하는 공간입니다.

## 규칙

- **팀장만** 이 디렉토리를 수정할 수 있습니다
- 일반 멤버는 자신의 `members/<본인-아이디>/` 디렉토리에서만 작업하세요
- 통합이 필요하면 팀장에게 요청하세요

## 통합 방법

1. 각 멤버가 본인 디렉토리에서 기능을 완성합니다
2. 팀장이 각 멤버의 코드를 검토합니다
3. 팀장이 이 디렉토리에 통합 코드를 작성합니다

## 디렉토리 구조

```
integrated/
├── README.md    ← 이 파일
└── src/         ← 통합 소스 코드
```
